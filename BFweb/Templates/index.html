Body Fat % Calculator

This BF% calculator has been created with machine learning by applying scikit-learn's Random Forest Regressor on a dataset of 253 men measuring age, weight, height, and 10 body circumferences. This dataset has been supplied by Dr. A. Garth Fisher and can be found here. Using an unseen 10% of this dataset, the trained model has been measured to have an r^2 value of .7730 and an mean absolute error(MAE) value of 2.819%
